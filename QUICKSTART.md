# 草果剪贴板 - 快速开始

## 🎉 恭喜！打包成功

您的草果剪贴板应用已成功打包！

## 📦 获取应用

### 应用程序位置
```
src-tauri/target/release/bundle/macos/草果剪贴板.app
```

### DMG 安装包位置
```
src-tauri/target/release/bundle/dmg/草果剪贴板_0.1.0_x64.dmg
```

**文件大小**: 4.0 MB

## 🚀 快速安装

### 推荐：使用 DMG 安装包

1. **双击** `草果剪贴板_0.1.0_x64.dmg`
2. **拖拽** 应用到"应用程序"文件夹
3. **启动** 应用程序

### 直接使用应用包

1. **双击** `草果剪贴板.app` 启动
2. 如果遇到安全提示，点击"打开"即可

## 📝 首次使用

### 1. 授予权限

首次启动时，系统可能会提示授予权限：

**辅助功能权限**（监听剪贴板需要）
```
系统设置 > 隐私与安全性 > 辅助功能
点击 "+" > 选择"草果剪贴板" > 勾选
```

### 2. 开始使用

1. **复制文本** (Cmd+C)
2. **呼出面板** (Cmd+Shift+V)
3. **选择记录** (↑↓ 或点击)
4. **复制内容** (Enter 或点击)

## ⌨️ 快捷键速查

| 快捷键 | 功能 |
|--------|------|
| `Cmd + Shift + V` | 呼出/隐藏面板 |
| `↑ / ↓` | 上下选择 |
| `Enter` | 复制选中 |
| `Esc` | 隐藏面板 |

## 💡 使用技巧

1. **自动监听**: 复制任何文本都会自动记录
2. **快速搜索**: 在面板顶部输入关键词即可搜索
3. **批量删除**: 搜索后可批量管理记录
4. **键盘操作**: 尽量使用快捷键，效率更高

## 🔧 常见问题

### Q: 快捷键不工作？
A: 确保应用正在运行，且已授予辅助功能权限。

### Q: 看不到历史记录？
A: 先复制一些文本，然后呼出面板查看。

### Q: 应用无法启动？
A: 检查系统是否为 macOS 10.15+，尝试重新安装。

### Q: 如何卸载？
A: 将应用拖到废纸篓，然后清空即可。
