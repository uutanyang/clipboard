# 保存图片到本地功能说明

## 新增功能

现在可以将剪贴板中的图片保存到本地文件系统了！

## 使用方法

### 方法一：使用保存按钮（推荐）

1. 复制一张图片到剪贴板（使用 `Cmd + Shift + 4` 截图）
2. 打开草果剪贴板应用
3. 鼠标悬停在图片项目上
4. 点击绿色的"保存"按钮（✓ 图标）
5. 图片将自动保存到下载目录

### 方法二：快速点击保存

1. 在图片项目上点击"保存"按钮
2. 确认保存成功提示
3. 应用会自动关闭

## 功能特点

✅ **自动命名**: 图片自动命名为 `screenshot_YYYYMMDD_HHMMSS.jpg`
✅ **智能格式检测**: 根据图片数据自动识别格式（JPEG/PNG/GIF/WebP）
✅ **默认目录**: 保存到系统下载目录（~/Downloads）
✅ **覆盖保护**: 如果文件名冲突，会使用时间戳避免覆盖
✅ **即时反馈**: 保存成功后自动关闭应用

## 保存位置

### macOS
```
~/Downloads/screenshot_20260216_161530.jpg
```

### Windows
```
C:\Users\你的用户名\Downloads\screenshot_20260216_161530.jpg
```

### Linux
```
~/Downloads/screenshot_20260216_161530.jpg
```

## 按钮说明

| 按钮 | 颜色 | 功能 |
|------|------|------|
| 保存按钮 | 绿色 | 将图片保存到本地下载目录 |
| 删除按钮 | 红色 | 删除该条剪贴板记录 |

## 快捷键

目前保存功能需要使用鼠标点击按钮。

快捷键说明：
- **Enter**: 复制选中的项目
- **↑↓**: 在列表中导航
- **Esc**: 关闭应用
- **鼠标悬停**: 显示保存和删除按钮

## 文件命名规则

### 默认格式
```
screenshot_YYYYMMDD_HHMMSS.jpg
```

### 示例
```
screenshot_20260216_161530.jpg  # 2026年2月16日 16:15:30
screenshot_20260216_161531.jpg  # 下一秒
```

### 自动格式检测

代码会根据图片数据的魔术字节自动检测格式：

| 格式 | 魔术字节 | 扩展名 |
|------|---------|--------|
| JPEG | FF D8 FF | .jpg |
| PNG | 89 50 4E 47 | .png |
| GIF | 47 49 46 | .gif |
| WebP | RIFF ... WEBP | .webp |
| 其他 | - | .jpg (默认) |

## 注意事项

1. **文件大小限制**: 建议单张图片不超过 50MB
2. **磁盘空间**: 确保下载目录有足够的磁盘空间
3. **权限问题**: 应用需要有写入下载目录的权限
4. **文件名冲突**: 使用时间戳自动避免冲突

## 故障排除

### 问题：保存失败

**可能原因：**
- 下载目录不存在或无权限
- 磁盘空间不足
- 图片数据损坏

**解决方法：**
1. 检查下载目录是否存在
2. 检查磁盘剩余空间
3. 查看控制台错误日志

### 问题：保存的图片打不开

**可能原因：**
- base64 数据损坏
- 格式识别错误

**解决方法：**
1. 确认图片在应用中可以正常显示
2. 尝试用不同的图片查看器打开
3. 查看控制台日志获取错误信息
